Additions
=========
Added NoClick to PiStorm32 version.
Added support for Pi4 emmc device to PiStorm32 version.
Added Croatian language to Non-English games.
Added new Amiga Game Selector release for ECS/OCS Amigas.
Added 'Boot to Game' mode to 'Expert Mode'.
Added the complete collection of WhatIFF? amiga guide magazines.
Added new options menu to manage WHDLoad save data.
Added PDF manuals to AMOS collection.
Added Bars 'n' Pipes (RTG Compatible)
Added option to set 'Expert Mode' on permanently until disabled.
Added device drivers drawer to the Storage folder.


Changes
=======
Changed Avail to MemClear in speed scripts.
Change size of Pistorm32 EMU68BOOT partition to 1gb.
Updated most text descriptions on main menu.
Removed MuTools from user-startup.
'Expert Mode' can now be set to stay on after reboot.
Enabled CDFS AutoMount in WinUAE version.
Removed intros and data disks from IGame data as most didn't launch.
Separated games and demos in random player script. No there are 2 entries in the
launcher.
Replaced tools daemon prefs with updated version.
Improved AGS splash screen.
Split random game/demo scripts.
Changed default theme.
Renamed 'Manuals & Cheats' menu to 'Add-on' Files.
Moved Readme files to Add-On list.
Rolled back MUI to v3.8
Centered window titles.
Re-structured A500 mini hdf files. 64gb Required.


Fixes
=====
Fixed random game script.
Fixed error in file log script.
Removed workbench library check.
Fixed 68040.library error.
Fixed 68030 black screen error.
Fixed Final Fight Enhanced.
Fixed Lionheart.
Fixed 'Games by Theme' creation script. Rampage is now correct version.
Fixed 'Restore Favourites' script.
Fixed IGame scripts on AGA Real Hardware version.
Fixed Midland Amiga Group Theme
Fixed corrupt graphics in Evil's Doom full version.
Fixed Another World Lost Levels
Fixed Robinson's Requiem
Fixed some Expert Mode script errors.
Fixed multiplayer games not adding to favourites.
