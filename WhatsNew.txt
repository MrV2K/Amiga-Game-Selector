                    Amiga Game Selector - What's New
                    ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~		 


Version v3.1 - Release xx-xx-26
===============================

Fixes
=====
01. Fixed patched scsi script on real hardware version.
02. Fixed cpu menu appearing in RHW & Retroarch versions.
03. Fixed SOFTIEEE breaking the RetroArch version on Mister.
04. Fixed Lionheart not showing main sprite.
05. Fixed some premium games running from launcher.
06. Fixed AMOS Pro not loading editors.
07. Fixed check_drives and start_ags scripts breaking expert mode.
08. Fixed font preference requesters on all versions of AGS.
09. Fixed Lock 'n Chase 020 naming which broke the launch script.
10. Fixed music not restarting after stopping boot mode.
11. Fixed Doom launching Doom II in premium games.
12. Fixed low volume on AHI system.
13. Fixed IGame_All script not finishing.
14. Fixed duplicate publishers in database.
15. Fixed missing archive sizes in AGS database.
16. Fixed missing install sizes in AGS database.
17. Fixed Mame & ScummVM assign not working if booting straight into launcher.
18. Fixed Carmageddon not showing graphics.
19. Fixed some typos in Start_AGS that stopped the CPU options being removed on
    RHW versions and some options being spelt wrong.
20. Fixed file requesters in Final Writer... again!
21. Fixed MechForce save paths in Extra Games.
22. Fixed ADF mounting on ECS version.
23. Fixed Kindwords 3 on ECS release.
24. Fixed "Show All" in filters.
25. Fixed Elf (Ocean) not showing in main launcher lists.
26. Fixed Scala MM400 install on UAE based versions of AGS.
27. Fixed missing archive version numbers in AGS database.
28. Fixed Ultimate Quiz 1 in 'Extra Games'
29. Fixed My Programs 'Add Program' and 'Instructions' options.
30. Fixed IGame images on real hardware AGA and Retroarch versions.
31. Fixed IGame loading on all versions.
32. Fixed Sinbad & the Throne of the Falcon speed.

Changes
=======
01. Combined the following hdfs reducing the number needed to be mounted.
		Workbench & Work
		WHD_Games & WHD_Demos
		Media & Music
		Games & Premium
02. Moved away from WBLoad as it caused crashes.
03. Changed file naming tag priority
04. Set Fast Copper for Scorpion Engine games on A500 mini and Amiberry.
05. Removed Mame from A500 Mini version as it doesn't have enough cpu speed
   and memory to run it properly.
06. Added english only gameslist to IGame.
07. Changed setpatch to v44.38.
08. Removed icon.library remlib from startup sequence.
09. Changed workbench.library v40.5 to default. Newer versions are in Libs:
10. Restructured and optimised startup-sequence and user-startup.
11. Added Pistorm and Retroarch variables to adapt scripts.
12. Updated HW select boot menu to run scripts rather than setting variables.
13. Separated Mame and ScummVM onto their own hdf partitons.
14. Switched on FPU JIT in the WinUAE config and fixed JIT_Enable script to
    automatically switch it on if disabled.
15. Remove MemClear from all scripts as it caused instabilities.
16. Changed default music mod in launcher.
17. Changed WHDLoad game icons to Workbench 3 style
18. Changed extra game icons to Workbench 3 style
19. Added ECS hardware variable to change some scripts.
20. Update ECS version to use DPaintV by default.
21. Updated compact flash drivers on all versions to v1.39
22. Added Pistorm DOSDrivers and startup script to ECS and AGA releases
23. Added DCTelnet to ECS release.
24. Updated Workbench 2.1 install with missing folders.
25. The Start_AGS script cleans up after itself a bit better.
26. Revisited the ZX Spectrum Collection. Fixed loads of errors and added 
    another 450 titles.
27. Updated the AGS OS folder to make a portable install easier.	
28. Updated WBDock on AGA version to v2.907
29. Added EaglePlayer to ECS release.
30. Increased the buffer on the PCMCIA card driver to stop crashes.
31. Changed WBDock to use no backdrops.
32. Removed Directory Opus PNG icon on AGA release.
33. Set Workbench screen depth to 16 colours on AGA / Retroarch release.
34. Added "Shut Down System" to Amiberry releases.
35. Added 'New & Updated' category to 'Games - By Theme' for the latest
    AGS game additions.
36. Removed 'Max Speed' option from the AGS database as it wasn't required.
37. Removed Dirk's themes by request.
38. Changed path in all AGS configs to not rely on assigns and make it easier
    to run in portable mode.
39. Improved playback on HippoPlayer on real hardware versions of AGS.
40. Set NOMMU and NOVBRMOVE in WHDLoad.prefs to help compatibility with 68060
    CPU's on real hardware systems.
41. Added a fast boot mode which removes the splash screens and significantly
    speeds up the boot process. Can be switched on and off from the launcher
	options and Workbench menu.
42. Removed SoftIEEE from AGS by default. Available in Workbench:Storage/Drivers.
43. Added subfolder support to the My Programs 'Add Programs' script.
44. Renamed 'Extra Games Premium' to 'Premium Games'
45. Added 'Extra Demos' category to main list.
46. Added 'New & Updated' category to the 'Games by Theme' list.
47. Standardised Flight Sim 2 names.
48. Added StackAttack to startup-sequence.
49. Added WhatIFF issue 5.20.
50. Improved navigation and menus in the launcher.
51. Moved boing ball demo and bad apple to Extra Demos.
52. Added Amiga Rulez Issue 2601.
53. Tidied up the WBDock launcher bar.
