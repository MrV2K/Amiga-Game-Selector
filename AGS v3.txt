Changes
=======
* Updated ECS WHDLoad executables.
* Moved AGS-Stuff script to new S:AGS drawer and renamed to ags.script on the
  emulated versions of AGS.
* Split the AGS HDD images to create modular system on emulated versions.
* Changed AGS scripts to use UnSet for environment variables rather than
  deleting them.
* Changed Reboot and UAEQuit to pause before acting so disk buffers can flush.
* Removed MCP as it broke on Kickstart 3.1.4 up.
* Moved Premium Games to main list.
* Moved expert mode check to before game boot mode to fix expert mode showing
  if it was disabled.
* Improved the sound quality on AmigaAmp
* Removed expired WarpDT datatype
* Removed 'Delete Program' from Expert Mode if the program has been added to
  favourites to avoid confusion.
* Removed adult games and demos from the random scripts.
* Hints & Cheats will now stop the launcher music on the A500 mini.
* Updated icon.library to the latest release. Added the archive contents to the
  drivers drawer.
* Music options scripts now detect whether a module is active before running.
* WHDLoad save backup script now allows you to copy the lha to a shared drive.
* WHDLoad restore script now allows you to delete the backup file.
* Moved Docs Disks to AGS/WHDLoad drive and made path local.
* All scripts now adjust for different HW variables.
* Changed "Pi" HW variable to "Amiberry"
* Updated the MMU libraries and added install folder to the drivers drawer.
